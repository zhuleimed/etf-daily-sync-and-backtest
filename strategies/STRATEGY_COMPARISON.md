# ETF 动量轮动策略 — 对比分析文档

> 最后更新: 2026-06-24
>
> 本目录记录所有已开发的策略回测结果，随新策略增加持续更新。
> 所有回测均使用统一数据源（etf_daily.db），统一费用模型（佣金0.02%、滑点0.01%），
> 统一 look-ahead 修正（信号用 idx-1，执行用 idx）。

## 1. momentum_rotation — 纯动量轮动

**定位：** 基准策略，所有过滤器的起点。

**逻辑：**
```
每日计算各 ETF 的 20日动量 → 排名 → 全仓第1名
```

| 参数 | 值 |
|------|----|
| MOMENTUM_WINDOW | 20 |
| MIN_SWITCH_CONVICTION | 3% |
| MIN_HOLD_DAYS | 10 |
| SHORT_TERM_MOMENTUM_CHECK | True |
| ADJUSTMENT_DAYS | 5 |
| RISK_MODE | A（无风控） |

**回测结果：**

| 年份 | 收益 | 说明 |
|------|------|------|
| 2024 | +18.76% | 震荡市中微盈 |
| 2025 | +39.97% | 捕捉创业板/科创50行情 |
| 2026(半年) | +29.52% | 延续强势 |

| 全周期指标 | 数值 |
|-----------|------|
| 累计收益 | +125.12% |
| 年化收益 | 41.01% |
| 最大回撤 | -25.80% |
| 夏普比率 | 1.06 |
| 回撤恢复天数 | ~365天 |
| 切换次数 | 19 |

## 2. momentum_ma_filter — 动量+大盘均线过滤

**定位：** 用沪深300均线判断大方向。

**逻辑：**
```
if HS300 > MA(HS300, N):  正常轮动
if HS300 ≤ MA(HS300, N):  全部空仓
```

| 参数 | 值 |
|------|----|
| MA_FILTER_PERIOD | 60/120/250 |

**回测对比：**

| MA周期 | 总收益 | 回撤 | 夏普 | 结论 |
|--------|--------|------|------|------|
| MA=60 | +87.43% | -25.65% | 0.88 | ❌ 频繁触发，错过涨幅 |
| MA=120 | +88.00% | -25.50% | 0.87 | ❌ 同上 |
| **MA=250** | **+124.80%** | **-28.63%** | **1.07** | ≈ 类似纯动量 |

**结论：** 在2024-2026牛市周期中，沪深300很少跌破年线（MA=250），熊市保护效果无法验证。短周期均线（60/120）频繁触发空仓，显著拖累收益。

## 3. momentum_dual — 双动量轮动

**定位：** 绝对动量+相对动量双重过滤。

**逻辑：**
```
if ETF_20d_momentum > 0 AND 排名第1:  持有
if 持仓ETF动量转负:                    卖出空仓
```

**回测结果：**

| 年份 | 收益 |
|------|------|
| 2024 | +13.17% |
| 2025 | +31.23% |
| 2026(半年) | +11.28% |

| 指标 | 数值 |
|------|------|
| 累计收益 | **+64.29%** |
| 最大回撤 | -29.38% |
| 夏普 | 0.70 |
| 切换次数 | 33 |

**结论：** 绝对动量（>0）条件在正常回调中频繁触发清仓，反复进出错过反弹，收益被大幅砍半。不推荐单独使用。

## 4. momentum_ma_etf — 动量+逐ETF均线过滤

**定位：** 每个ETF独立用自身均线做买入门槛。

**逻辑：**
```
if close[ETF] > MA(ETF, N):  进入候选池
if close[ETF] ≤ MA(ETF, N):  排除/卖出
候选池中按动量排名选最强
```

**回测对比：**

| MA周期 | 总收益 | 回撤 | 夏普 | 结论 |
|--------|--------|------|------|------|
| MA=60 | +102.02% | -28.37% | 0.95 | ❌ 空仓29%时间，错过涨幅 |
| MA=120 | +77.60% | -27.80% | 0.78 | ❌ 更差 |
| MA=250 | +125.12% | -25.80% | 1.06 | ≈ 同纯动量 |

**结论：** 与 momentum_ma_filter 类似，短周期均线在牛市中过度触发，长周期均线（250）从未触发。无实质改进。

## 5. momentum_vol_filter — 波动率过滤轮动 ✅ 夏普最优

**定位：** 用沪深300历史波动率判断市场状态，高波动时空仓。

**逻辑：**
```
年化波动率 = std(daily_return, 20) × √252
if 年化波动率 < 30%:  正常轮动
if 年化波动率 ≥ 30%:  全部空仓（高波动→趋势不稳→不做）
```

| 参数 | 值 |
|------|----|
| VOL_THRESHOLD | 30%（年化） |
| VOL_WINDOW | 20日 |
| ADJUSTMENT_DAYS | 3（有波动率保护，快速切换更安全） |

**全周期结果：**

| 年份 | 收益 |
|------|------|
| 2024 | +14.65% |
| 2025 | **+50.18%** |
| 2026(半年) | **+35.42%** |

| 指标 | 数值 |
|------|------|
| 累计收益 | **+131.88%** |
| 年化收益 | 42.79% |
| 最大回撤 | **-24.86%** |
| 夏普比率 | **1.23** |
| 回撤恢复天数 | **155天** |
| 切换次数 | 19 |

**波动率阈值敏感性测试：**

| 阈值 | 总收益 | 夏普 | 特征 |
|------|--------|------|------|
| 25% | +62.6% | 0.79 | ❌ 太敏感 |
| 28% | +65.7% | 0.82 | ❌ 仍敏感 |
| **30%** | **+131.9%** | **1.23** | ✅ 最佳 |
| 32% | +124.6% | 1.19 | ≈ 稳定区 |
| 35% | +124.6% | 1.19 | ≈ 稳定区 |

## 7. 关键经验总结

### 7.1 过滤器的核心问题

在 2024-2026 年这一整体上行周期中，**任何基于价格位置的过滤器（均线/绝对动量）都会频繁触发**，导致空仓时间过长，错过修复性反弹。

**均线过滤的问题（monentum_ma_filter / momentum_ma_etf）：**
- MA=60/120 → A股60日均线在震荡市中反复穿越 → 频繁空仓
- MA=250（年线）→ 牛市中几乎不触发 → 等同纯动量

**绝对动量过滤的问题（momentum_dual）：**
- 动量>0条件在正常回调中轻易转负 → 过早清仓
- 等动量转正时已错过反弹

### 7.3 双均线交叉轮动的局限性

双均线交叉（MA交叉）在所有测试组合中均大幅落后动量策略：
- 最佳组合 MA(60,120) 仅 +44.5%，夏普 0.81
- 切换次数极高（79-96次），说明信号不稳定
- 均线交叉天然滞后——确认上升趋势时行情已过半，确认下降时已跌了一段
- 在 2024-2026 年的震荡上行市中反复被左右打脸

### 7.2 为什么波动率过滤成功

波动率过滤的核心是**衡量"市场是否适合做动量"**，而不是"市场在涨还是跌"：
- 低波动→趋势清晰→动量有效→正常轮动
- 高波动→趋势不稳→动量频繁反转→空仓回避

**关键区别：**均线/绝对动量在牛市中频繁"假警报"，波动率触发次数少且精准。

### 7.3 非动量策略测试

**低波动率轮动（low_vol_rotation）：**
- 持有过去20日年化波动率最低的ETF
- +32.36%，切换50次，但夏普仅0.58
- 在2024-2026年的上行市中，低波动=低收益，错过了大部分涨幅

**均值回归轮动（mean_reversion）：**
- 持有距20日均线最远的ETF（乖离率为负且绝对值最大），预期超跌反弹
- +80.21%，但切换136次（成本7.26%），夏普0.94
- 频繁进出说明均值回归信号不稳定，在趋势市中反复被打脸

**结论：两种非动量策略均未超越动量策略。** 均值回归在震荡市中可能有效，但2024-2026年的整体上行趋势不利于该策略。

### 7.4 成交量/通道/布林带策略

**量价配合轮动（vol_price_momentum）：**
- 信号 = 动量 × 成交量放大倍数（放量上涨才追）
- +55.30%，但切换181次，成本9.3%，夏普0.71
- 成交量放大条件过于严格，降低了有效信号数量

**唐奇安通道突破轮动（donchian_breakout）：**
- 价格突破60日最高价买入，跌破60日最低价卖出
- +67.40%，切换108次，成本5.4%，夏普0.85
- 突破策略在趋势市有效，但2024-2026的震荡上行中频繁假突破

**布林带轮动（bollinger_rotation）：**
- 价格在布林带中的位置打分：靠近下轨（超卖）→买入；靠近上轨（超买）→卖出
- +108.13%，但切换217次，成本12.5%，夏普1.10
- 夏普及格高但换手率过高

### 7.5 组合策略（动量+配对）

**配对交易（pair_trading）— 多空版本（需融券）：**
- z-score均值回归多空对冲策略，市场中性
- **关键优化：**删除高相关性的重叠配对（如上证50↔沪深300），改用不同风格的ETF（大盘vs成长）
- 最终配置：上证50↔创业板 + 上证50↔科创50，周期60日，阈值2.0
- **优化后结果：+20.09%，29次交易，最大回撤仅-7.13%**
- 配对交易的成功关键在于选择真正会发生分歧的标的，而非同质化ETF

**配对交易 → 纯多头风格轮动（2026-06 新增，无需融券）：**
- 发现数据库无融券数据，原多空策略不可执行
- 改为纯多头切换：3对 z-score 中取 |z| 最大的信号，全仓切换至"便宜方"ETF
- 开仓阈值从 2.0 提升至 3.0（减少噪音信号，夏普提升 0.95→1.22）
- ZSCORE_CLOSE=0.3, ZSCORE_PERIOD=60（与原版一致）
- **完整区间结果：+128.75%，夏普 1.19，最大回撤 10.14%（2023.03→2026.06）**
- **逐年收益：2023+4.41% / 2024+66.66% / 2025+28.07% / 2026+2.65%（全部正收益）**
- 不对称阈值测试确认对称 3.0 为最优

### 7.4 经验法则

1. **不要用价格位置做过滤器**（MA、绝对动量）— 在牛市中它们频繁出清，得不偿失
2. **波动率是最有效的市场状态衡量指标** — 它衡量的是"趋势的可信度"而非"涨跌方向"
3. **过滤器不应损害收益** — 真正好的过滤器只在最需要的时候触发
4. **T+1 交易逻辑** — 信号用当日收盘、执行用次日开盘，避免 look-ahead bias
